A Black, Dark, & Light set of icons for use with the EmbyIcons Plug-In with two themes (MaterialCard and MaterialBanner).

These were created mostly for my use, but I'm happy to share. I urge you to check back for releases as I learn of more icons and how EmbyIcons uses them. I'm still learning how the plug-in's Icon Manager works and how to interpret results as several reported "missing" items are never used by it.

My main goals in creating them:

- Readability on televisions and displays.
- Matching Emby's Font and UI as close as possible.
- Colors that would compliment Emby's. Several badges are based on accent colors available in Emby.
- Use material design icons, again, going with Emby's usage of them.

# Installation

The release contains all three versions.

1. Click the Latest release here.
2. Download the Source Code format of your choice (zip or tar.gz).
3. Extract your download. Inside the Release directory, you'll find the MaterialIcons folder which has folders for each of the three themes.
4. Put them location accessible to Emby/EmbyIcons and select which to use in the plug-in's preferences.

# Issues and Troubleshooting

- If you don't like one of the icons, just remove it from the folder. EmbyIcons will fall back to its built-in icon for that item or nothing will be displayed depending on your settings.
- If you find any icons missing, please open an issue here and I'll generate a new release. Please look in EmbyIcons' Icon Manager and *Scan Library & Icons*. I will need to know what the icon badge is supposed to say and the filename that it expects it to have.
- There are extra icons in the packs. When you do a *Scan Library & Icons* with the plug-in, it may return icons that are unique to your media. My pack may have icons that your media doesn't have and visa-versa.
- Future releases will continue to refine which are actually needed. Other than a small amount of space for extra icons, it does not hurt to have a few files that may not be used.
